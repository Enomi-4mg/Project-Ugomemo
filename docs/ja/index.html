Project Ugomemo

Project Ugomemo は、ピクセル風のフレームアニメーションを描き、再生確認し、音声と同期して書き出すためのデスクトップ制作ツールです。

現在の最新テストリリースは v0.1.5-test です。GitHub Releases から macOS Apple Silicon向けDMG、Windows x64向けEXE/MSI を入手できます。

Download latest test build GitHub README

Draw

ペン、ブラシ、トーン、消しゴム、シェイプを使い、3レイヤーと固定パレットで硬いピクセル表現を作れます。

Playback

フレーム列を確認しながら、FPSと速度を切り替えてアニメーションの動きをすばやく検証できます。

Audio

素材追加、録音、4トラック配置、ミキサー、フレーム同期再生で音声つき作品を組み立てられます。

Concept

Project Ugomemo は、1枚描いて、次のフレームを作り、すぐ再生し、音声まで合わせるための小さなアニメーション制作環境です。

汎用的なプロ向け制作ソフトではなく、軽く、直接的に、短いフレームアニメーションを試作できるワークステーションを目指しています。

Features

描く、動きを見る、音を合わせる、保存する、書き出す、という制作の流れに沿って機能を整理しています。

Drawing

Pen, Brush, Tone, Eraser, Shape

Animation

フレーム単位編集、ページ/フレーム列、オニオンスキン

Layers

A / B / C の3レイヤー

Playback

FPSセレクター、速度調整、プレビュー再生

Audio

音声素材追加、録音、4トラック配置、ミキサー、フレーム同期

File

.upj プロジェクト保存/読込

Export

画像、動画、音声つき動画の書き出し

Basic Workflow

  1. 新規プロジェクトを作成する
  2. Draw Mode でフレームを描く
  3. Playback Mode で動きを確認する
  4. Audio Mode で音声素材や録音を配置する
  5. 画像・動画・音声つき動画として書き出す

Tech Stack

Tauri のデスクトップアプリとして、React UI と Rust 側のネイティブ処理を組み合わせています。

  • Tauri
  • React
  • TypeScript
  • Rust
  • Web Audio API
  • ffmpeg sidecar export flow

Project File

Project Ugomemo は .upj 形式でプロジェクトを保存します。 .upj には、フレーム画像、レイヤー情報、プロジェクト設定、Audio Mode のタイムライン状態が含まれます。

詳細な仕様や開発者向け情報は README を参照してください。
