Pixel Animation Workstation

Project Ugomemo

ピクセル風アニメーション制作アプリ。

GitHub Releases

Draw

ペン、ブラシ、トーン、消しゴム、シェイプを使い、3レイヤーと固定パレットで硬いピクセル表現を作れます。

Playback

フレーム列を確認しながら、FPSと速度を切り替えてアニメーションの動きをすばやく検証できます。

Audio

素材追加、録音、4トラック配置、ミキサー、フレーム同期再生で音声つき作品を組み立てられます。

Notes

デスクトップアプリのプロジェクトファイルは .upj を使います。 ネイティブの保存/読込ダイアログ、音声素材の同梱、画像書き出し、Tauri sidecarによる動画/音声書き出しに対応します。

このGitHub Pagesページは docs/ から配信できる静的ページです。 アプリ本体の白いグリッド、ピンクのコントロール、モノスペース文字、強い罫線のリズムに合わせています。
