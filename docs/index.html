Pixel Animation Workstation

Project Ugomemo

A pixel-style animation production app.

GitHub Releases

Draw

Use pen, brush, tone, eraser, and shape tools with three layers and a fixed palette to create crisp pixel artwork.

Playback

Review the frame rail, switch FPS and speed, and quickly check the motion of your animation.

Audio

Add materials, record audio, arrange four tracks, mix levels, and sync playback to frame timing.

Notes

Desktop app project files use .upj. The app supports native save/load dialogs, bundled audio assets, image export, and video/audio export through the Tauri sidecar flow.

This GitHub Pages site is static and can be served from docs/. It mirrors the app's white grid, pink controls, mono type, and high-contrast panel rhythm.
